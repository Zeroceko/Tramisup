Activate systematic debugging for: $ARGUMENTS

1. Gather evidence: logs, error messages, stack traces, reproduction steps
2. Form hypotheses about root cause
3. Test each hypothesis methodically
4. Identify root cause with evidence
5. Implement and verify the fix
