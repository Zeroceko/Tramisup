# ASO Advisor

**Primary use:** Tiramisup should use this skill to generate user-facing App Store Optimization guidance for founders shipping mobile apps.

Use this skill when recommending how users should improve app title, subtitle, keywords, screenshots, preview video, localized listing copy, icon, conversion, and store-page messaging. This skill is for **listing optimization and discoverability**, not for legal, policy, or submission-compliance advice.

## Use This For

Apply this skill when the task involves any of the following:
- app store optimization strategy
- ASO keyword and metadata guidance
- app title / subtitle / short description improvement
- screenshot order and conversion messaging
- listing copy for App Store or Google Play
- localized store-page copy
- conversion-oriented store-page recommendations

## Do Not Use This For

- privacy policy, terms, subscription compliance, or review-rejection guidance
- internal engineering deployment checklists
- generic SEO advice for websites unless the user explicitly asks for app listing optimization

## Core Principle

ASO is about **discoverability + conversion**. Help users get found in the store and make the store page convince the right users to install.

## Required Recommendation Areas

When creating ASO guidance for users, make sure recommendations cover these groups.

### 1. Metadata Strategy

Always check whether the user has:
- a clear, searchable app title
- a subtitle / short description that explains the benefit fast
- keywords or search terms that match real user intent
- category/store positioning aligned with the app’s core use case

### 2. Visual Conversion

Always check whether the user has:
- screenshots that show the core value quickly
- screenshot order that tells a story, not random UI
- a preview video only if it improves clarity
- an icon that is distinct, readable, and consistent with the product
- assets sized and cropped for the target store surfaces
- the first screenshot doing the heaviest conversion work

### 3. Message Match

Always check whether the listing copy matches:
- the actual shipped experience
- the main problem the app solves
- the audience’s vocabulary
- the product stage and promise level

### 4. Localization and Market Fit

Do not forget to recommend:
- localized title/subtitle/copy when relevant
- market-specific wording if the audience is not English-first
- store-page language that matches the user’s actual acquisition channel

## Output Style

When using this skill for user-facing guidance:
- group recommendations by priority
- separate **must-fix for listing clarity** from **should-improve for conversion**
- explain *why* each item matters in one sentence
- keep recommendations practical and founder-friendly
- avoid generic “growth hacks”; be specific about listing assets and user intent

## Preferred Priority Structure

### Critical for Listing Clarity
Items that block store-page understanding or search relevance.

### Strongly Recommended
Items that improve install conversion and store discoverability.

### Nice to Improve
Items that sharpen positioning but are not urgent.

## Common ASO Heuristics

Flag these as high-risk when advising users:
- title too generic to search
- screenshots that do not tell the app’s main story
- screenshots that bury the core value in later frames
- subtitle/description that sounds like marketing fluff
- keyword stuffing or unnatural phrase repetition
- icon not readable at small size
- listing copy that promises features the app does not yet have

## How Tiramisup Should Use This Skill

If Tiramisup is generating guidance for users, prefer outputs like:
- ASO audit summary
- listing optimization checklist
- title/subtitle rewrite suggestions
- screenshot sequence recommendations
- localized store-page improvement plan

Do not present this as guaranteed ranking advice. Present it as practical store-page optimization guidance.
